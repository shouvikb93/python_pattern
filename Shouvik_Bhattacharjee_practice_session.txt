Q. Given a year, determine whether it is a leap year. If it is a leap year, return the Boolean True, otherwise return False. (Use a function)

Note that the code stub provided reads from STDIN and passes arguments to the is_leap function. It is only necessary to complete the is_leap function                                 (hackerrank)

Soln.

def is_leap(year):
    leap = False

    if year % 400 == 0:
        leap = True
    elif year % 100 == 0:
        leap = False
    elif year % 4 == 0:
        leap = True

    return leap

year = int(input())
print(is_leap(year))

Q. membership operators

Soln.

numbers = [10, 20, 30, 40, 50]

print(20 in numbers)
print(25 in numbers)

print(20 not in numbers)
print(25 not in numbers)

**** 'in' operator exists in other languages like javascript but they are not used in the same manner as that used in python. in python, in operator is used as membership operator and used to check values in a list, tuple etc. but in other languages say for example javascript, in operator is used to check keys/properties (not values).
